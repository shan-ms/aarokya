# Aarokya Design System

## Brand Identity
- Name: Aarokya (आरोग्यम्) — "wholistic wellbeing"
- Tagline: "Your Pocket Hospital"
- Logo: Smile+ (healthcare smile with plus sign)
- Voice: Warm, caring, accessible, trustworthy

## Color Palette
- Primary: #2563EB (Trust Blue)
- Secondary: #10B981 (Health Green)
- Accent: #F59E0B (Warmth Amber)
- Error: #EF4444
- Background: #F8FAFC
- Surface: #FFFFFF
- Text Primary: #1E293B
- Text Secondary: #64748B

## Typography
- Headings: Inter Bold
- Body: Inter Regular
- Vernacular: Noto Sans (language-specific variants)

## Design Principles
1. **Care-first**: Every interaction should feel like someone asking "Are you ok?"
2. **Accessible**: Low-literacy friendly, icon-heavy, voice-enabled
3. **Trustworthy**: Transparent finances, no dark patterns
4. **Multilingual**: 12+ Indian languages, RTL-ready
5. **Low-bandwidth**: Optimized for 2G/3G connections

## Key Screens (Customer App)
1. Onboarding: language + "who is this for" → phone → OTP → intent selection → minimal profile → consent notice → ABHA link (optional) → progressive health profile
2. Home: "Are you okay?" check-in CTA, HSA balance, quick actions (consult, book, records), trust strip
3. Check-in: guided symptom triage → self-care / book consult / emergency guidance
4. HSA Detail: balance, contributions timeline, goal tracker
5. Insurance: eligibility meter, coverage options, active policies
6. Records vault: upload/scan/tag medical documents, consent-based sharing
7. Health: profile, medications, reminders
8. Contributions: source breakdown, monthly trends
9. Privacy center: consent management, data download, account deletion

## Key Screens (Partner App)
1. Dashboard: enrolled workers, total contributions, impact metrics
2. Workers: list, search, add, individual HSA status
3. Contribute: bulk/individual, amount config, payment
4. Reports: CSR reports, compliance docs, analytics

## Key Screens (Control Center)
1. Dashboard: system health, user growth, financial overview
2. Users: search, detail view, verification workflow
3. Finances: transactions, reconciliation, fraud alerts
4. Insurance: policies, claims, provider management
5. Analytics: charts, funnels, geographic heat maps
6. Settings: roles, permissions, system config
